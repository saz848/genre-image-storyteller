Introduction

Team members: Sarah Ahmad & Lisa Cox

This is a neural network that generates stories based on photos. It works by first captioning the photo and then generating a story based on that caption. We want to see how well the system is able to reflect different genres of text input when given pictures of the same genre. This can be a tool for creating short stories in different genres based on an image, or generating genre-focused captions for things like Instagram photos.
